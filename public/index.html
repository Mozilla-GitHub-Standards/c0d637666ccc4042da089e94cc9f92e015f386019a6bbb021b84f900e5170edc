WebVR

  • Download Firefox Nightly: https://nightly.mozilla.org/
    Open Nightly and install the WebVR Add­on: http://mozvr.com/downloads/
    If you are using an Oculus Rift...
    Install the latest runtime: https://developer.oculus.com/
    Configure your displays per: http://mozvr.com/downloads/
    Restart Nightly and visit http://mozvr.com/projects/sechelt/. You should see a fullscreen "goggles" icon. Click on it to enter VR mode! Press escape to exit VR mode.
  • For technical primer, see Mozilla engineer Vlad Vukicevic's original WebVR blog post:
    http://blog.bitops.com/blog/2014/06/26/first­steps­for­vr­on­the­web/
    To start building WebVR experiences, it's easiest to start with WebVR boilerplates:
    https://github.com/mozvr/
    https://github.com/borismus/webvr­boilerplate/
    https://github.com/povdocs/webvr­starter­kit
    For deeper details on the WebVR API and it's implementation, we recommend:
    https://developer.mozilla.org/en­US/docs/Web/API/WebVR_API
    http://mozvr.github.io/webvr­spec/webvr.html
  • The Oculus Rift DK2 is currently the best­supported headset for WebVR development. Limited support for mobile browsers is possible using responsive boilerplates like https://github.com/borismus/webvr­boilerplate,but the Rift has the best combination of capabilities (eg positional tracking), performance (eg resolution) etc. Support for additional headsets like the HTC Vive is being worked on.
  • Make sure to follow the VR setup tips at http://mozvr.com/downloads/. VR is a very new technology, and a few settings can make the difference between a good or bad experience.
